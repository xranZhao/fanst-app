# 书库托管选阿里云 OSS，不用 Cloudflare R2 和拾光坞 NAS

3000+ 本小说（913MB，85% 为 RAR 压缩包）需要被手机 PWA 按需拉取。决定：阿里云 OSS（标准-本地冗余，公共读 Bucket）+ `index.json` 索引；拾光坞 NAS 只做备份底稿，通过电脑端同步脚本回灌。

**原因**：PWA 需要免登录、允许 CORS、稳定直链的 HTTP 访问。

- **拾光坞 NAS（否决）**：消费级私有云，文件访问依赖其 App/网页登录，分享链接经中转服务，没有稳定公开直链和 CORS 保证，外网速度不可控。
- **Cloudflare R2（否决）**：功能合适且免费额度够，但开通强制绑定外币信用卡或 PayPal，用户没有外币卡；且 r2.dev 临时域名在国内访问稳定性无保障。
- **阿里云 OSS（选中）**：国内访问快、支持公共读 + CORS、用户已有实名认证账号和余额。估算月成本约 ¥0.3（存储 2GB × ¥0.12 + 极少流量），现有余额可用十年以上。

**考虑过的其他备选**：GitHub Pages 直接放书库（超 1GB 站点上限，否决）；腾讯云 COS（与 OSS 等价，因用户在阿里云有余额而落选）。
